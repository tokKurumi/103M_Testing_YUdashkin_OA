# equation.txt - Checklist for quadratic equation solver
# Format: a, b, c => expected result (description)

# Test Cases:

1. a=1, b=-3, c=2
   Expected: (1.0, 2.0)
   Description: Two distinct real roots in ascending order

2. a=1, b=2, c=1
   Expected: (-1.0,)
   Description: One real root (double root, discriminant=0)

3. a=1, b=0, c=1
   Expected: ()
   Description: No real roots (discriminant < 0)

4. a=0, b=2, c=-4
   Expected: (2.0,)
   Description: Linear equation (a=0): 2x - 4 = 0

5. a=2, b=-7, c=3
   Expected: (0.65..., 2.29...)
   Description: Two real roots with non-unity leading coefficient

6. a=1, b=0, c=-9
   Expected: (-3.0, 3.0)
   Description: Two real roots symmetric about origin

7. a=0, b=0, c=1
   Expected: ValueError (not an equation)
   Description: Invalid: both a and b are zero
