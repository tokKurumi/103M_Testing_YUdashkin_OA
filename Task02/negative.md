# Чек-лист негативных сценариев функционального тестирования

## Некорректные действия с клетками

| № | Проверка | Ожидаемый результат |
|---|----------|---------------------|
| 1 | Повторный клик по уже открытой клетке | Клетка не изменяется, никаких побочных эффектов не возникает |
| 2 | Правый клик (установка флага) по открытой клетке | Флаг не устанавливается, клетка остаётся без изменений |
| 3 | Долгое нажатие (400+ мс) на открытую клетку на мобильном | Флаг не устанавливается, клетка остаётся без изменений |
| 4 | Левый клик по клетке с установленным флагом | Клетка не открывается, флаг остаётся на месте (защита от случайного открытия) |
| 5 | Попытка клика за пределами игрового поля | Никаких изменений в игре не происходит, ошибок не возникает |
| 6 | Множественные быстрые клики по одной клетке | Клетка обрабатывается один раз, повторные клики игнорируются |
| 7 | Множественные быстрые клики по разным клеткам | Каждая клетка обрабатывается корректно, состояние игры не нарушается |
| 8 | Одновременный левый и правый клик по закрытой клетке | Приложение не зависает, действие обрабатывается предсказуемо |

## Действия после завершения игры

| № | Проверка | Ожидаемый результат |
|---|----------|---------------------|
| 9 | Клик по клетке после поражения | Клетка не открывается, игровое поле заблокировано |
| 10 | Клик по клетке после победы | Клетка не открывается, игровое поле заблокировано |
| 11 | Установка флага на клетку после завершения игры | Действие не выполняется |
| 12 | Попытка поставить паузу после завершения игры | Пауза не активируется, кнопка неактивна или действие игнорируется |

## Некорректные действия с сохранениями

| № | Проверка | Ожидаемый результат |
|---|----------|---------------------|
| 13 | Ввести пустое название при сохранении | Сохранение не выполняется, отображается предупреждение |
| 14 | Ввести название длиннее 256 символов | Текст обрезается до 256 символов или отображается предупреждение о превышении лимита |
| 15 | Попытка загрузить сохранение при отсутствии сохранённых игр | Отображается пустой список или сообщение об отсутствии сохранений |
| 16 | Ручная модификация данных в localStorage (повреждённые данные) | Приложение корректно обрабатывает повреждённые данные без критических ошибок |
| 17 | Очистка localStorage во время игры (через DevTools) | Приложение продолжает работать, автосохранение восстанавливается при следующем ходе |
| 18 | Попытка сохранить игру после завершения (поражение/победа) | Действие не выполняется или сохранение отклоняется |
| 19 | Загрузка сохранения с несуществующей конфигурацией поля | Приложение обрабатывает ошибку, предлагает начать новую игру |
| 20 | Ввести название сохранения со спецсимволами (<, >, &, ", ') | Название сохраняется корректно, спецсимволы экранируются, XSS не возникает |

|---|----------|---------------------|
| 21 | Попытка возобновить игру, не находясь на паузе | Действие игнорируется |
| 22 | Многократное быстрое нажатие на кнопку «Пауза» | Состояние паузы переключается корректно, таймер не сбивается |
| 23 | Свернуть браузер и развернуть через длительное время | Таймер корректно отражает прошедшее время |
| 24 | Перевод системных часов назад во время игры | Таймер продолжает работать корректно, не сбрасывается и не показывает отрицательное время |

|---|----------|---------------------|

## Мобильные устройства

| № | Проверка | Ожидаемый результат |
|---|----------|---------------------|
| 32 | Случайное короткое касание (менее 400 мс) при намерении установить флаг | Клетка открывается (короткое касание = клик), а не устанавливается флаг |
| 33 | Свайп по игровому полю | Прокрутка страницы или игровое поле, клетки случайно не открываются |
| 34 | Поворот устройства во время игры | Игровое поле корректно перерисовывается, состояние игры не сбрасывается |
| 35 | Многопальцевое касание (pinch-to-zoom) на игровом поле | Масштабирование обрабатывается без случайного открытия клеток |

## Граничные значения

| № | Проверка | Ожидаемый результат |
|---|----------|---------------------|
| 36 | Установить флаги на все клетки поля | Приложение не ограничивает количество флагов и не вызывает ошибок |
| 37 | Открыть клетку в углу поля (минимум соседей) | Число отображается корректно (только для имеющихся соседей) |
| 38 | Открыть клетку на краю поля | Число отображается корректно с учётом отсутствия части соседей |
| 39 | Таймер доходит до 99:59 (максимум для формата mm:ss) | Таймер корректно отображает время, не вызывает ошибок форматирования |
| 40 | Попытка сохранить игру при полностью заполненном localStorage | Отображается сообщение об ошибке, приложение не падает |
