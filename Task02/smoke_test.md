# Чек-лист дымового тестирования (Smoke Test)

Минимальный набор проверок для подтверждения работоспособности основной функциональности приложения и пригодности сборки к дальнейшему тестированию.

| № | Проверка | Ожидаемый результат |
|---|----------|---------------------|
| 1 | Загрузка приложения в браузере | Страница открывается без ошибок, отображается экран загрузки, затем главное меню |
| 2 | Выбор уровня сложности «Новичок» и начало игры | Генерируется поле 8×8, отображаются счётчики (время: 00:00, мины: 10), игра готова к первому ходу |
| 3 | Открытие клетки на игровом поле (первый клик) | Клетка безопасна (не мина), при 0 соседних мин выполняется рекурсивное открытие, таймер начинает отсчёт |
| 4 | Установка и снятие флага на закрытой клетке | Правый клик ставит флаг, повторный клик снимает; счётчик флагов обновляется |
| 5 | Поражение — открытие клетки с миной | Игра завершается, все мины видны, открытая мина выделена красным, отображается экран поражения |
| 6 | Победа — открытие всех безопасных клеток | Игра завершается, таймер останавливается, отображается экран победы |
| 7 | Кнопка «Новая игра» после завершения партии | Генерируется новое поле на текущем уровне, счётчики сбрасываются |
| 8 | Сохранение и загрузка игры | Игра сохраняется через диалог, загружается из списка сохранений с полным восстановлением состояния |
| 9 | Отображение статистики | Панель статистики открывается, показывает количество игр, побед, поражений |
| 10 | Переключение темы (светлая ↔ тёмная) | Цветовая схема интерфейса изменяется, выбор сохраняется после перезагрузки |
