# Чек-лист позитивных сценариев функционального тестирования

## Игровая логика

| № | Проверка | Ожидаемый результат |
|---|----------|---------------------|
| 1 | Открыть клетку без мины с числом (1-8) левым кликом | Клетка открывается и отображает количество соседних мин |
| 2 | Открыть пустую клетку (0 соседних мин) левым кликом | Клетка открывается и автоматически рекурсивно открываются все смежные пустые клетки до клеток с числами включительно |
| 3 | Первый клик по полю попадает на клетку | Клетка безопасна (не содержит мину), вокруг неё (радиус 1 клетки) мин нет |
| 4 | Открыть клетку с миной | Наступает поражение: все мины становятся видны, открытая мина выделена красным |
| 5 | Открыть все безопасные клетки (не содержащие мин) | Наступает победа, отображается сообщение о победе |
| 6 | Установить флаг на закрытую клетку правым кликом | На клетке появляется флаг |
| 7 | Снять флаг с помеченной клетки правым кликом | Флаг убирается, клетка снова в состоянии «закрытая» |
| 8 | Установить флаг на закрытую клетку долгим нажатием (400+ мс) на мобильном | На клетке появляется флаг |
| 9 | Снять флаг долгим нажатием (400+ мс) на мобильном | Флаг убирается, клетка снова в состоянии «закрытая» |
| 10 | Открыть клетку одиночным касанием на мобильном устройстве | Клетка открывается аналогично левому клику |
| 11 | Рекурсивное открытие пустых клеток не затрагивает клетки с флагами | Клетки, помеченные флагами, остаются закрытыми при автооткрытии соседних пустых клеток |
| 12 | Победа без установки флагов на все мины | Достаточно открыть все безопасные клетки — установка флагов на мины не требуется |

## Уровни сложности

| № | Проверка | Ожидаемый результат |
|---|----------|---------------------|
| 13 | Выбрать уровень «Новичок» | Генерируется поле 8×8 с 10 минами |
| 14 | Выбрать уровень «Любитель» | Генерируется поле 16×16 с 40 минами |
| 15 | Выбрать уровень «Профессионал» | Генерируется поле 16×30 с 99 минами |
| 16 | Начать новую игру на текущем уровне кнопкой «Новая игра» | Генерируется новое поле того же размера и с тем же количеством мин |
| 17 | После завершения игры выбрать другой уровень | Отображается меню выбора уровня, при выборе начинается новая игра на выбранном уровне |

## Таймер

| № | Проверка | Ожидаемый результат |
|---|----------|---------------------|
| 18 | Таймер до первого клика по полю | Таймер не запущен, отображается 00:00 |
| 19 | Первый клик по игровому полю | Таймер начинает отсчёт в формате mm:ss |
| 20 | Победа в игре | Таймер останавливается, итоговое время отображается на экране результата |
| 21 | Поражение в игре | Таймер останавливается |
| 22 | Нажатие кнопки «Пауза» | Таймер приостанавливается, отображается экран паузы |
| 23 | Нажатие кнопки «Продолжить» после паузы | Таймер возобновляет отсчёт с момента паузы |
| 24 | Превышение максимального времени (10 мин для «Новичка») | Игра автоматически завершается поражением |
| 25 | Превышение максимального времени (40 мин для «Любителя») | Игра автоматически завершается поражением |
| 26 | Превышение максимального времени (99 мин для «Профессионала») | Игра автоматически завершается поражением |

## Сохранение и загрузка

| № | Проверка | Ожидаемый результат |
|---|----------|---------------------|
| 27 | Автосохранение после каждого хода | Состояние игры записывается в localStorage (поле, статусы клеток, время, номер хода) |
| 28 | Перезагрузка страницы при незавершённой игре | Предлагается продолжить незавершённую игру |
| 29 | Согласиться продолжить незавершённую игру | Игра восстанавливается с сохранённым состоянием поля, временем и ходами |
| 30 | Ручное сохранение через кнопку «Сохранить прогресс» | Открывается диалог ввода названия сохранения |
| 31 | Ввести название и подтвердить сохранение | Игра сохраняется в localStorage под указанным именем |
| 32 | Открыть экран «Загрузить сохранение» | Отображается список всех сохранённых игр |
| 33 | Выбрать сохранение и загрузить его | Игра восстанавливается с сохранённым состоянием |
| 34 | Удалить сохранение (клик на иконку корзины) | Появляется попап подтверждения с кнопками «Отмена» и «Удалить» |
| 35 | Подтвердить удаление сохранения | Сохранение удаляется из localStorage и списка |
| 36 | Отменить удаление сохранения (кнопка «Отмена») | Сохранение остаётся, попап закрывается |

## Счётчики и информация

| № | Проверка | Ожидаемый результат |
|---|----------|---------------------|
| 37 | Счётчик мин при начале игры | Отображается общее количество мин для текущего уровня |
| 38 | Счётчик флагов при установке флага | Счётчик флагов увеличивается на 1 |
| 39 | Счётчик флагов при снятии флага | Счётчик флагов уменьшается на 1 |
| 40 | Счётчик ходов при открытии клетки | Счётчик ходов увеличивается на 1 |

## Статистика

| № | Проверка | Ожидаемый результат |
|---|----------|---------------------|
| 41 | Открыть панель статистики | Отображается общая статистика: всего игр, побед, поражений, процент побед |
| 42 | Статистика по уровням | Для каждого уровня отображается количество побед и лучшее время |
| 43 | Обновление статистики после победы | Количество побед увеличивается, лучшее время обновляется (если текущее лучше) |
| 44 | Обновление статистики после поражения | Количество поражений увеличивается |
| 45 | Отображение даты последней игры | Дата отображается в формате dd.mm.yyyy (ru) или mm/dd/yyyy (en) |
| 46 | Очистка статистики | При подтверждении («Вы уверены? Это действие необратимо.») вся статистика сбрасывается |
