# Чек-лист функционального тестирования

## 1. Игровое поле и генерация

| № | Проверка | Ожидаемый результат |
|---|----------|---------------------|
| 1 | Генерация поля уровня «Новичок» | Создаётся сетка 8×8 (64 клетки), на поле размещено ровно 10 мин |
| 2 | Генерация поля уровня «Любитель» | Создаётся сетка 16×16 (256 клеток), на поле размещено ровно 40 мин |
| 3 | Генерация поля уровня «Профессионал» | Создаётся сетка 16×30 (480 клеток), на поле размещено ровно 99 мин |
| 4 | Все клетки при начале игры в состоянии «Закрытая» | Ни одна клетка не открыта, флагов нет |
| 5 | Мины распределяются случайно | При повторных запусках одного уровня расположение мин различается |
| 6 | Первый клик безопасен | Первая открытая клетка не содержит мину |
| 7 | Вокруг первой открытой клетки нет мин (радиус 1) | 8 соседних клеток (или меньше на краях) вокруг первого клика не содержат мин |

## 2. Механика открытия клеток

| № | Проверка | Ожидаемый результат |
|---|----------|---------------------|
| 8 | Открытие клетки с числом (1) | Клетка отображает цифру «1» синим цветом |
| 9 | Открытие клетки с числом (2) | Клетка отображает цифру «2» зелёным цветом |
| 10 | Открытие клетки с числом (3) | Клетка отображает цифру «3» красным цветом |
| 11 | Открытие клетки с числом (4) | Клетка отображает цифру «4» тёмно-синим цветом |
| 12 | Открытие клетки с числом (5) | Клетка отображает цифру «5» тёмно-красным цветом |
| 13 | Открытие клетки с числом (6) | Клетка отображает цифру «6» голубым цветом |
| 14 | Открытие клетки с числом (7) | Клетка отображает цифру «7» чёрным (светлая тема) / белым (тёмная тема) цветом |
| 15 | Открытие клетки с числом (8) | Клетка отображает цифру «8» серым цветом |
| 16 | Открытие пустой клетки (0 мин вокруг) | Клетка открывается пустой, запускается рекурсивное открытие соседних клеток |
| 17 | Рекурсивное открытие останавливается на клетках с числами | Клетки с числами отображаются и открываются, но не вызывают дальнейшее рекурсивное открытие |
| 18 | Рекурсивное открытие не затрагивает клетки с флагами | Клетки с флагами остаются закрытыми при автооткрытии |
| 19 | Открытие клетки с миной | Наступает поражение |

## 3. Механика флагов

| № | Проверка | Ожидаемый результат |
|---|----------|---------------------|
| 20 | Установка флага правым кликом на закрытой клетке | Появляется значок флага |
| 21 | Снятие флага правым кликом на помеченной клетке | Флаг убирается, клетка возвращается в состояние «Закрытая» |
| 22 | Установка флага долгим нажатием (≥ 400 мс) на мобильном | На клетке появляется флаг |
| 23 | Снятие флага долгим нажатием (≥ 400 мс) на мобильном | Флаг убирается |
| 24 | Флаг невозможно поставить на открытую клетку | Правый клик по открытой клетке не производит никаких действий |
| 25 | Открытую клетку с числом левый клик не изменяет | Повторный клик не вызывает действия |

## 4. Условия победы и поражения

| № | Проверка | Ожидаемый результат |
|---|----------|---------------------|
| 26 | Все безопасные клетки открыты | Игра завершается победой, отображается экран «Вы победили!» |
| 27 | Победа без установки всех флагов | Достаточно открыть все клетки без мин — флаги не обязательны |
| 28 | Таймер останавливается при победе | Время фиксируется, отображается итоговое значение |
| 29 | Статистика обновляется при победе | Счётчик побед увеличен, лучшее время обновлено при необходимости |
| 30 | Открытие мины приводит к поражению | Экран «Вы проиграли!» с сообщением «Вы открыли мину» |
| 31 | При поражении все мины становятся видны | Все мины на поле отображаются |
| 32 | Мина, вызвавшая поражение, выделена красным | Открытая пользователем мина имеет красный фон (`#ff4444` / `#c62828`) |
| 33 | Таймер останавливается при поражении | Отсчёт времени прекращается |
| 34 | Статистика обновляется при поражении | Счётчик поражений увеличен на 1 |

## 6. Сохранение и загрузка

| № | Проверка | Ожидаемый результат |
|---|----------|---------------------|
| 42 | Автосохранение записывает конфигурацию поля | В localStorage сохраняется размер поля, расположение мин, сид генератора |
| 43 | Автосохранение записывает статус каждой клетки | Состояние «открыта/закрыта» и «флаг/без флага» для каждой клетки |
| 44 | Автосохранение записывает текущее время и номер хода | Время и номер хода доступны при восстановлении |

| 47 | Название сохранения — максимум 256 символов | Ограничение на длину вводимого текста |
| 48 | Загрузка сохранённой игры из списка | Игра полностью восстанавливается: поле, время, ходы, флаги |
| 49 | Удаление сохранения с подтверждением | Попап подтверждения: «Удалить сохранение?», кнопки «Отмена» и «Удалить» |
